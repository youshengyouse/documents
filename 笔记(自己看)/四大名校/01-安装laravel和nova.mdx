## 安装 laravel 脚手架

- 项目名：sidamingxiao-20250709

```bash
# 当前目录 /www2024/1_doing
cp /www2024/0_common/php/laravel/laravel-20250709/ sidamingxiao-20250709 -r
cd sidamingxiao-20250709
cp .env.example .env
composer install
php artisan key:generate
```

## 修改 composer.json 配置

```bash

composer config minimum-stability dev
# 默认 "minimum-stability": "stable",

composer config repositories.laravel '{"type":"path","url":"/www2024/0_common/php/laravel/**/","options":{"symlink":true}}'
composer config repositories.yousheng '{"type":"path","url":"/www2024/0_common/php/yousheng/**","options":{"symlink":true}}'
```
