## 简易的命令行入门教程:

Git 全局设置:

```bash
git config --global user.name "chinabangong"
git config --global user.email "19936926974@189.cn"
```

创建 git 仓库:

```bash
mkdir source_laravel_laravel
cd source_laravel_laravel
git init
touch README.md
git add README.md
git commit -m "first commit"
git remote add origin https://gitee.com/chinabangong/source_laravel_laravel.git
git push -u origin "master"
```
