## 安装

### Fumadocs 官方

```bash
# 确保不用国内镜像，否则会导致安装失败
root@pan:/www2024/5_docs# pnpm config list
registry=https://registry.npmjs.org/ # 这里是npm的镜像
# 下面全程开启科学上网
# 当前目录 /www2024/5_docs
# git clone git@github.com:fuma-nama/fumadocs.git
git clone githubyoushengyouse:fuma-nama/fumadocs.git official
# 上面在省图书馆时，开启科学上网反而慢，改为不科学上网
root@pan:/www2024/5_docs/fumadocs/official# pnpm i
root@pan:/www2024/5_docs/fumadocs/official# pnpm run build --filter=./packages/*
root@pan:/www2024/5_docs/fumadocs/official# pnpm dev  --filter=docs
```
