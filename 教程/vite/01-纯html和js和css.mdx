## script>标签的type属性
- text/javascript（默认值）
- module（即type="module"）：这个属性值是 ES6 模块的专用标识，它能让 JavaScript 文件通过import和export语句实现模块化加载。和普通脚本比起来，它有这些特点：
- - 脚本会以严格模式（'use strict'）运行。
- - 可以使用import和export语句。
- - 默认采用延迟加载（defer），也就是会等到 DOM 解析完成后再执行。
- - 模块内部的变量和函数具有局部作用域，不会污染全局作用域。
- text/esm(等同module)
- importmap
- module-shim
- text/babel 或 application/javascript

## 总结
- 建议设置 type="module"，现在浏览器都支持。
- 建议使用 import 语句，而不是 script 标签的 src 属性。
- package.json 中的 type 字段如果设置为 module。或文件名为.mjs
- import 默认不支持"main.js"这种写法，相对路径或绝对路径，扩展名.js不能省略(安装vite后可以)
- 默认浏览器不支持.ts
