## 导航菜单

- 参考：https://www.radix-ui.com/primitives/docs/components/navigation-menu 效果很好，可惜不是 tailwindcss，不是很方便
- 参考：https://flowbite.com/docs/components/navbar/ 展开收缩箭头没变化，小遗憾
- 参考: https://flyonui.com/docs/navigations/navbar/# mega menu 可以考虑
- 参考：https://tailwindcss.com/plus/ui-blocks/marketing/elements/flyout-menus 其实不错，只是箭头没变化
- 参考：https://www.material-tailwind.com/docs/react/navbar# 【初步确定这个】
- 参考：https://ui.shadcn.com/docs/components/navigation-menu【最后使用这个】
- https://floatui.com/components/navbars 这里的也不错
- https://www.curateduilist.com/components/navigation-menu 集成常见的导航菜单

## 安装

```bash
pnpm dlx shadcn@latest add navigation-menu
```
