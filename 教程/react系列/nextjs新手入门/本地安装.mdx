## 第一步：本地准备 nextjs 源码

```bash
git clone -b canary --single-branch https://github.com/vercel/next.js.git nextjs-20250717
cd nextjs-20250717
git checkout -b wang origin/canary
pnpm install
pnpm dev
```

## 第三步：切换为本地的 nextjs

删除 node_modules,package.lock 目录，重新安装依赖。

```bash
# pnpm add /home/tom/www2026/0_common/js/nextjs-20250717/{packages/next,node_modules/{react,react-dom}}
pnpm add /home/tom/www2026/0_common/js/nextjs-20250717/packages/next
```
