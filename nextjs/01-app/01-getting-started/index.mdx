本**入门指南**部分将帮助您创建第一个Next.js应用，并学习每个项目中都会用到的核心功能。

## 预备知识

我们的文档假设您对Web开发有一定了解。在开始之前，建议您熟悉以下内容：

- HTML
- CSS
- JavaScript
- React

如果您是React新手或需要复习，我们建议从[React基础课程](/learn/react-foundations)开始，以及[Next.js基础课程](/learn/dashboard-app)，该课程将引导您边学边构建一个应用程序。
