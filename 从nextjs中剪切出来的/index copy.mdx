欢迎访问Next.js文档！

## 什么是Next.js？

Next.js是一个用于构建全栈Web应用的React框架。您可以使用React组件构建用户界面，并通过Next.js获得额外的功能和优化。

它还会自动配置底层工具如打包器和编译器。您可以专注于产品开发并快速交付。

无论您是独立开发者还是团队一员，Next.js都能帮助您构建交互式、动态且快速的React应用。

## App路由和Pages路由

Next.js有两种不同的路由：

- **App路由**：支持Server Components等新React特性的新路由
- **Pages路由**：原始路由，仍在维护和改进中

在侧边栏顶部，您会看到一个下拉菜单，可以在[App路由](/docs/app)和[Pages路由](/docs/pages)文档间切换

## 预备知识

我们的文档假设您对Web开发有一定了解。开始前最好熟悉：

- HTML
- CSS
- JavaScript
- React

如果您是React新手或需要复习，我们建议从[React基础课程](/learn/react-foundations)开始，以及[Next.js基础课程](/learn/dashboard-app)，边学边构建应用
